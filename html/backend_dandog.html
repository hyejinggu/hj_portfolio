커뮤니티(게시판) 관련 페이지

라운지 및 이벤트 페이지

--- 라운지 ---

사용자들이 자유롭게 자신의 의견을 나누는 공간(4개의 하위 카테고리로 구성)

검색, 정렬, 페이지 네이션 구현

검색의 경우 글 내용, 글 제목, 제목+내용, 작성자 등으로 검색 가능

검색 조건의 변수가 다양하여 Criteria API 활용

로그인 했을 경우에만 하단의 글쓰기 버튼이 노출

작성자 ID, 로그인 ID를 비교하여 본인이 쓴 글일 경우 '수정|삭제', 다른 사람일 경우 '추천' 버튼 표시

조회수 증가, 중복 추천 방지 기능 구현 (시연 영상은 아래 '마이 페이지'에서 확인 가능)

--- 이벤트 ---

현재 날짜와 비교하여 DB에 저장된 이벤트 종료 날짜와 비교하여 종료된 이벤트와 진행 중인 이벤트를 다르게 표시

우리 동네 페이지

현재 위치 정보를 불러온 뒤 현재 위치 주변의 검색어가 포함된 상점들을 화면에 표시 (Kakao API 활용)

화면 하단 좌측 - 상점의 평균 별점과 리뷰 개수를 표시

화면 하단 우측 - 지도 위의 상점 마커 클릭 또는 좌측의 상점 이름 클릭 시 상세한 후기 표시

검색 조건 - 기본 가나다순, 평균 별점순, 리뷰 개수 순

평균 별점은 3.5, 4.0, 4.5 이상으로 세부 정렬 가능

마이 페이지 - 내가 쓴 글 관리

커뮤니티(라운지, 우리 동네)에 쓴 글을 마이페이지에서 바로 확인하여 수정, 삭제 가능

조회수 증가, 중복 추천 방지, 글 수정|삭제 기능 확인 가능

관리자 페이지

관리자 상품 페이지

상품 등록

관리자 아이디로 로그인 할 경우 헤더의 네비게이션 바를 통해 관리자 페이지로 이동

동일한 상품이지만 컬러, 사이즈가 다른 경우 각 상품 옵션별 재고 관리가 필요하므로 상품마다 고유한 번호 부여

관리자의 상품 등록 편의를 위해 상품 등록 시 컬러, 사이즈를 다중 선택하여 일괄 등록이 가능하도록 구성

대표사진 1, 2는 각각 사진을 하나씩 등록하게 되며 옵션 상관없이 모든 상품에 부여됨.

상세 이미지는 ITEM_IMAGES 테이블을 하나 더 만들어 따로 관리

불필요한 데이터 낭비 방지를 위해 동일한 이름의 상품 중 고유번호가 가장 작은 상품에만 상세 이미지 등록

상품 검색 및 수정

Pageable을 활용한 페이지 네이션 구현

전체, 카테고리별 + 상품 이름, 상품 번호로 검색 가능

상품 이름 클릭 시 상품 상세 정보 확인, 상품 정보 수정 가능

상품 상세 이미지 수정 시 기존의 이미지 + 새로운 이미지를 자유롭게 선택하여 수정하도록 구성

관리자 코드 페이지

쇼핑몰 내 코드 관리

쇼핑몰 내에서 사용하는 컬러, 사이즈 정보를 관리할 수 있도록 구성

기존의 컬러, 사이즈 정보 확인이 가능하며 원하는 컬러와 사이즈 등록 가능

필요할 경우 새로운 코드 정보를 등록 가능
